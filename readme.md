# 293 Miniclub - MINI Countryman Webseite

## Projektbeschreibung

Dies ist eine Mockup-Webseite für den MINI Club, erstellt von Marco Fritsche. Die Seite präsentiert den MINI Countryman und weitere MINI Modelle mit ihren Besonderheiten und Vorteilen.

## Inhalte

Die Webseite bietet Informationen über:

- **MINI Countryman** - Der vielseitigste MINI mit Platz für fünf Erwachsene
- **Gepäckraum** - Bis zu 1.390 Liter Fassungsvermögen
- **Freihändiges Öffnen** - Easy Opener mit Fussbewegung
- **ALL4 Allradantrieb** - Dynamische Leistungsverteilung
- **MINI Electric** - Elektrifizierter Fahrspass
- **MINI 5 Door** - Mehr Platz bei gewohntem MINI Feeling
- **MINI Cabrio** - Fahrspass unter freiem Himmel

## Installation & Verwendung

1. Repository klonen oder herunterladen
2. `index.html` in einem Webbrowser öffnen
3. Durch die Seite navigieren und Inhalte erkunden

Alternativ kann die Seite auch auf einem Webserver gehostet werden.
